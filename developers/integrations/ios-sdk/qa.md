# 常见问题

### 1. 集成后运行报错 <a id="ios_1"></a>

确认一下 依赖的埋点SDK是否是2.8.7以上  
  
Xcode 的控制台打印可看 **集成SDK版本  
GrowingIO version: 2.8.7 !!!  
Thank you very much for using GrowingIO Touch. SDK version 1.1.2**

### 2.为什么 iOS 收不到推送消息？ <a id="ios_1"></a>

确认集成步骤是否缺失

确认app是否开启推送

确认证书与包名都正确

确认选择生产环境 和开发环境正确，

确认证书没有过期

**3.SDK最低兼容iOS系统。**

最低兼容iOS 8.0 系统。
